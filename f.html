生成超链接

a元素的局部属性: href hreflang media rel target type

属性 说明
href 指定a元素所指资源的URL
hreflang 说明所链接资源使用的语言
media 说明所链接资源用于哪种设备.
rel 说明文档与所链接资源的关系类型
target 指定用于打开链接资源的浏览环境
type 说明所链接资源的MIME属性,如 text/html

为文字添加下划线

u元素让一段文字从周围内容中凸显出来,但并不表示强调或其重要性有所增加.

u元素以前只具有呈现方面的作用(为文字加上下划线),没有实际的语义.即呈现性元素,其效果就是为文字添加下划线.

不要随便用CSS来篡改u元素的这种用途.

在样式方面,跟a元素表现一样,为防止混淆,应尽量避免使用u元素.

使用语言元素

有5个HTML元素的用途是为使用非西方语言提供支持的.

注音符号(ruby character)是用来帮助读者掌握表意语言文字正确发音的符号,位于这些文字上方或右方.

ruby元素表示一段包含注音符号的文字.

ruby元素需要和rt元素和rp元素搭配使用.rt元素用来标注注音符号,rp元素用来标记供不支持注音符号特性的浏览器,显示在注音符号前后的括号.

测试样例如下:

魑(chī) 魅(mèi) 魍(wǎng) 魉(liǎng)

快速制作 --- 文字转拼音在线汉字转拼音带声调
